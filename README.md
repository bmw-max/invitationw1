# Wedding Invitation - Deployment

This folder contains the production-ready files for the wedding invitation.

## How to Deploy to GitHub Pages

1.  Create a new repository on GitHub.
2.  Upload the `index.html` file from this folder to the root of your repository.
3.  Go to the repository **Settings** > **Pages**.
4.  Under **Build and deployment**, ensure the source is set to **Deploy from a branch**.
5.  Select your branch (usually `main`) and the root folder `/ (root)`.
6.  Click **Save**.

Your invitation will be live at `https://<your-username>.github.io/<your-repo-name>/`.
